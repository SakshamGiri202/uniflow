# UniFlow — One Login, Infinite Campus Possibilities 🎓

**UniFlow** is a unified digital ecosystem designed to solve the fragmentation of the Indian campus experience. By providing a verified, student-only environment, UniFlow empowers students to trade essentials, register for nationwide events, and share knowledge without the interference of middlemen or untrusted outsiders.

---

## 🚀 Key Features

### 🛒 Student Marketplace (Zero Middleman)
- **Safe Trading:** Buy and sell hostel essentials (books, cycles, electronics) exclusively within the student community.
- **Affordability:** Direct P2P transactions eliminate extra costs.

### 📅 Event Central
- **Discovery:** Find cultural, technical, and academic fests across universities.
- **Easy Registration:** One-click registration for inter and intra-college activities.

### 🧠 Knowledge Exchange
- **Peer Learning:** A structured platform for students to host workshops and mentor others.
- **Skill Building:** Collaborative spaces to learn new technologies and hobbies.

---

## 🛠️ Tech Stack

- **Frontend:** Reactjs/ Nextjs / Java
- **Backend:** Supabase
- **Database:** PostgreSQL
- **Auth:** Clerk auth

---

## 🛡️ Security First
UniFlow uses a robust verification system to ensure that every user is an active student, maintaining the integrity and safety of the marketplace and collaborative spaces.
